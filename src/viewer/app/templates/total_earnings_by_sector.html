This visualisation shows the total earnings for each sector over time. Each trading day all companies in a given sector are summed:

  • for each company: EPS * number of shares listed is computed
  • only companies with a positive EPS will contribute to each sector
  • both EPS and number of shares are obtained from the ASX data API
  • timeframe: {{ timeframe.description }}
  • all sectors results over the timeframe are shown below
  • as this is not a per-share measure you can more readily see changes in total earnings across sectors
